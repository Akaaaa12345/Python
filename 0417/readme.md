# 📋 리스트 자료구조 (List)

여러 개의 데이터를 하나의 변수에 순서대로 모아서 관리하는 파이썬의 가장 대표적인 자료구조를 정리한 공간입니다.

---

### 1. 리스트의 기본 개념
* **개념:** 대괄호(`[]`)를 사용하며, 여러 가지 형태의 데이터(숫자, 문자열 등)를 콤마(`,`)로 구분하여 하나의 상자에 담는 것
* **인덱싱 (Indexing):** 리스트 안의 방 번호는 **0번부터 시작**함 (뒤에서부터 셀 때는 `-1`번부터 시작)
* **슬라이싱 (Slicing):** `list[시작:끝]` 형태로 리스트의 일부분을 잘라서 가져옴 (끝 번호 바로 직전까지만 가져옴)



### 2. 리스트 주요 함수 및 연산
* **추가:** `.append(값)` (맨 뒤에 추가), `.insert(인덱스, 값)` (원하는 위치에 삽입)
* **삭제:** `del 리스트[인덱스]` (해당 방 데이터 삭제), `.remove(값)` (특정 값을 찾아서 삭제)
* **정렬:** `.sort()` (오름차순 정렬), `.reverse()` (순서 뒤집기)
* **기타:** `len(리스트)` (리스트의 전체 요소 개수 반환)
