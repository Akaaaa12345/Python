📦 함수 (Function)

특정 기능을 수행하는 코드들을 하나로 묶어 재사용 가능하게 만드는 함수의 핵심 개념을 정리한 공간입니다.

---

### 1. 함수의 기본 개념 및 장점
* **개념:** 프로그램 안의 작은 프로그램 (특정 입력값을 받아 처리하고 결과값을 돌려주는 상자)
* **장점:** * 중복 코드를 제거하여 코드가 깔끔해짐 (코드의 재사용성)
  * 에러가 났을 때 해당 함수만 고치면 되므로 유지보수가 편해짐



### 2. 함수의 구조 (정의와 호출)
* **함수 정의 (`def`):** `def 함수이름(매개변수):` 형태로 새로운 함수를 만듦
* **함수 호출:** 정의한 함수 이름과 필요한 값을 넣어 실제로 실행시킴
* **인수(Argument) vs 매개변수(Parameter):**
  * **매개변수:** 함수를 정의할 때 값을 받는 변수 (함수 내부의 변수)
  * **인수:** 함수를 호출할 때 함수에 실제로 전달하는 값

### 3. 반환값(Return)과 변수의 범위
* **`return` 문:** 함수가 실행을 마치고 자신을 호출한 곳으로 **결과값을 돌려줄 때** 사용 (값을 반환하고 함수는 즉시 종료됨)
* **지역 변수 (Local Variable):** 함수 내부에서 만들어져 **함수 안에서만** 사용할 수 있는 변수
* **전역 변수 (Global Variable):** 함수 외부(프로그램 전체)에서 만들어져 **어디서나** 접근할 수 있는 변수
