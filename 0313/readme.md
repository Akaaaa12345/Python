🔢 변수(Variable)와 수식(Expression)

데이터를 저장하는 공간인 **변수**와 이를 계산하는 **수식**의 기본 개념을 정리한 공간입니다.

---

### 1. 변수 (Variable): 데이터 저장 공간
* **개념:** 값을 담는 상자 (`변수명 = 값` 형태로 대입)
* **특징:** 자료형(정수, 실수, 문자열 등)을 파이썬이 자동으로 판단
* **명명 규칙:** 영문/숫자/_ 활용 (단, 숫자로 시작 불가, 예약어 사용 불가)



### 2. 수식과 연산자 (Expression & Operator)
* **산술 연산자:** `+` (더하기), `-` (빼기), `*` (곱하기), `/` (나누기)
* **특수 연산자:** `//` (몫), `%` (나머지), `**` (거듭제곱)
* **복합 대입 연산자:** `+=`, `-=`, `*=` (기존 값에 누적 연산)
* **우선순위:** 괄호 `()` 내부 > 거듭제곱 > 곱셈/나눗셈 > 덧셈/뺄셈
