# 🔢 변수, 수식, 그리고 입력(Input)

데이터를 입력받고, 저장하고, 계산하는 프로그램의 가장 기본적인 흐름을 정리한 공간입니다.

---

### 1. 데이터 입력 (Input): 사용자에게 값 받기
* **`input()` 함수:** 사용자로부터 키보드로 값을 입력받는 함수
* **주의점:** `input()`으로 입력받은 모든 값은 기본적으로 **문자열(str)**로 인식됨
* **형변환:** 숫자로 계산하려면 `int(input())`(정수)이나 `float(input())`(실수)으로 감싸야 함

### 2. 변수 (Variable): 데이터 저장 공간
* **개념:** 값을 담는 상자 (`변수명 = 값` 형태로 대입)
* **특징:** 자료형(정수, 실수, 문자열 등)을 파이썬이 자동으로 판단
* **명명 규칙:** 영문/숫자/_ 활용 (단, 숫자로 시작 불가, 예약어 사용 불가)

### 3. 수식과 연산자 (Expression & Operator)
* **산술 연산자:** `+` (더하기), `-` (빼기), `*` (곱하기), `/` (나누기)
* **특수 연산자:** `//` (몫), `%` (나머지), `**` (거듭제곱)
* **우선순위:** 괄호 `()` 내부 > 거듭제곱 > 곱셈/나눗셈 > 덧셈/뺄셈
