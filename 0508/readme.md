
🗂️ 자료구조 (Tuple, Dictionary, Set) & 문자열 (String)

파이썬에서 제공하는 다양한 자료구조의 특징과 문자열을 효율적으로 다루는 방법을 정리한 공간입니다.

---

### 1. 튜플 (Tuple): 수정할 수 없는 리스트
* **특징:** 소괄호(`()`)를 사용하며, 리스트와 거의 같지만 **한 번 생성하면 값을 변경하거나 삭제할 수 없음**
* **용도:** 프로그램이 실행되는 동안 절대로 바뀌지 않아야 하는 고정된 데이터(예: 캘린더의 요일, 좌표 등)를 안전하게 보관할 때 사용

### 2. 딕셔너리 (Dictionary): 사전형 자료구조
* **특징:** 중괄호(`{}`)를 사용하며, **`Key(키): Value(값)`**의 쌍으로 데이터를 관리함
* **용도:** 데이터의 순서(인덱스) 대신 의미 있는 이름(Key)으로 값을 찾고 싶을 때 사용 (단, Key 값은 중복될 수 없음)
* **예시:** `phone_book = {"홍길동": "010-1234-5678"}`

### 3. 세트 (Set): 집합 자료구조
* **특징:** 중괄호(`{}`)를 사용하며, 수학의 집합처럼 **중복을 허용하지 않고 순서가 없음**
* **용도:** 리스트 등에서 중복된 데이터를 싹 제거하거나, 교집합(`&`), 합집합(`|`), 차집합(`-`) 같은 집합 연산이 필요할 때 활용

### 4. 문자열 (String)
* **특징:** 문자들의 나열로, 리스트처럼 인덱싱과 슬라이싱을 통해 원하는 글자만 쏙 골라낼 수 있음 (튜플처럼 원본 수정은 불가능)
* **주요 함수:** * `.upper()` / `.lower()` : 대소문자 변환
  * `.split()` : 특정 기준(공백 등)으로 문자열을 쪼개서 리스트로 만듦
  * `.replace(A, B)` : 문자열 안의 A를 B로 변경
