# 🔀 조건문 (Conditional Statement)

주어진 조건에 따라 프로그램의 실행 흐름을 제어하는 조건문의 핵심 개념을 정리한 공간입니다.

---

### 1. 조건문(if)의 기본 개념
* **개념:** "만약 조건이 참(True)이면 코드를 실행하고, 거짓(False)이면 건너뛴다!"
* **들여쓰기(Indentation):** 파이썬은 조건문 아래 실행할 코드를 반드시 **들여쓰기(공백 4칸 또는 Tab)**로 구분함 (안 하면 에러 발생)



[Image of if else statement flowchart]


### 2. 조건문의 3가지 구조
* **`if` 문:** 조건이 참일 때만 특정 코드를 실행
* **`if - else` 문:** 조건이 참일 때와 거짓일 때 실행할 코드를 각각 나누어 처리
* **`if - elif - else` 문:** 여러 개의 조건을 차례대로 검사하여 해당하는 블록만 실행 (다중 조건 처리)

### 3. 비교 연산자와 논리 연산자
* **비교 연산자:** `==` (같다), `!=` (다르다), `>`, `<`, `>=`, `<=`
* **논리 연산자:**
  * `and`: 두 조건이 모두 참이어야 참
  * `or`: 두 조건 중 하나만 참이어야 참
  * `not`: 참을 거짓으로, 거짓을 참으로 뒤집음
