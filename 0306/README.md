## 🖨️ print() 함수 핵심 정리

`print()`는 파이썬에서 가장 기본적이고 자주 쓰이는 **화면 출력 함수**입니다. 컴퓨터에게 "이 내용을 모니터 화면에 보여줘"라고 명령하는 역할을 합니다.

### 1. 문자열(Text) 출력 규칙
* **따옴표 필수:** 글자나 문장을 출력할 때는 반드시 양옆을 **큰따옴표(`""`)나 작은따옴표(`''`)**로 감싸야 합니다.
* **에러 주의:** 따옴표를 생략하거나 한쪽만 쓰면 `SyntaxError`(문법 오류)가 발생합니다.
```python
print("안녕하세요")          # 올바른 예시
print('파이썬은 강력합니다.') # 올바른 예시
# print(안녕하세요)         # 에러 발생 (따옴표 누락)
