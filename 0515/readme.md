
# 🏗️ 객체(Object)와 클래스(Class)

코드를 독립적인 단위로 나누어 관리하는 **객체 지향 프로그래밍(OOP)**의 핵심 개념을 정리한 공간입니다.

---

### 1. 클래스와 객체의 개념 (붕어빵 틀과 붕어빵)
* **클래스 (Class):** 객체를 만들기 위한 **'설계도' 또는 '붕어빵 틀'**입니다. 데이터(속성)와 기능(메서드)을 정의만 해둔 상태입니다.
* **객체 (Object) / 인스턴스 (Instance):** 설계도(클래스)를 바탕으로 **실제 메모리에 만들어진 '결과물' 또는 '붕어빵'**입니다. 하나의 클래스로 여러 개의 서로 다른 객체를 찍어낼 수 있습니다.



### 2. 클래스의 구성 요소: 속성과 메서드
* **속성 (Attribute):** 객체가 가지고 있는 **'상태'나 '데이터'**를 말하며, 클래스 안의 **변수**로 표현됩니다. (예: 자동차의 색상, 현재 속도)
* **메서드 (Method):** 객체가 할 수 있는 **'행동'이나 '기능'**을 말하며, 클래스 안의 **함수**로 표현됩니다. (예: 가속하기, 브레이크 밟기)

### 3. 파이썬 클래스 핵심 문법
* **`class` 키워드:** 새로운 클래스를 정의할 때 사용합니다.
* **생성자 (`__init__`):** 객체가 생성될 때 **자동으로 호출되는 특별한 메서드**로, 객체의 속성(초기값)을 설정할 때 사용합니다.
* **`self` 키워드:** 생성되거나 실행 중인 **'객체 자신'**을 가리키는 파이썬만의 규칙입니다. 메서드를 정의할 때 첫 번째 매개변수로 반드시 적어주어야 합니다.

```python
# 클래스 정의 예시
class Car:
    def __init__(self, color, speed):
        self.color = color   # 속성
        self.speed = speed   # 속성

    def drive(self):         # 메서드
        print(f"{self.color} 자동차가 {self.speed}km/h로 달립니다.")

# 객체 생성 및 사용
my_car = Car("빨간색", 60)
my_car.drive()  # 출력: 빨간색 자동차가 60km/h로 달립니다.
