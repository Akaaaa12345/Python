# 🔁 반복문 (Loop)

정해진 조건이나 횟수에 따라 코드를 반복 실행하여 프로그램의 효율을 극대화하는 반복문의 핵심 개념을 정리한 공간입니다.

---

### 1. 반복문의 두 가지 종류
* **`for` 문 (횟수 제어 반복):** 대상을 지정하거나 정해진 횟수만큼 깔끔하게 반복할 때 사용
* **`while` 문 (조건 제어 반복):** 특정 조건이 참(True)인 동안 끝없이 반복할 때 사용 (조건이 거짓이 되면 탈출)



### 2. 반복문의 핵심 파트너
* **`range()` 함수:** `for` 문과 함께 쓰이며 연속된 숫자들을 생성해 줌
  * `range(5)`: 0부터 4까지 (총 5번)
  * `range(1, 10, 2)`: 1부터 9까지 2씩 건너뛰며 생성
* **무한 루프(Infinite Loop):** `while` 문의 조건이 계속 참일 때 발생하며, 프로그램을 종료하려면 `Ctrl + C`를 눌러야 함

### 3. 반복문의 흐름 제어 (Break & Continue)
* **`break`:** 반복문이 실행 중이더라도 즉시 그 자리에서 **반복문을 완전히 탈출**함
* **`continue`:** 남은 코드를 실행하지 않고, **다음 반복 순서로 바로 패스(건너뛰기)**함
